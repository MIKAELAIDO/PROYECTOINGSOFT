# PROYECTOINGSOFT
REPOSITORIO PROYECTO CARPETAS INFO
Definicion del problema:
La pérdida de objetos personales como credenciales, botellas y cargadores es una situación frecuente. La ausencia de un sistema centralizado para reportar y localizar estos artículos dificulta su recuperación, provocando pérdidas económicas, desperdicio de tiempo y frustración entre los estudiantes. Actualmente, la información sobre objetos perdidos suele dispersarse en grupos de redes sociales, lo que reduce significativamente las probabilidades de encontrar al propietario o al objeto extraviado.

Resumen ejecutivo:
En la institucion educativa es común que los estudiantes extravíen objetos personales como credenciales, objetos personales como cargadores, etc. ,  generando pérdidas de tiempo y gastos innecesarios. Actualmente no existe un medio centralizado y eficiente para reportar y recuperar estos artículos. El Sistema de Objetos Perdidos y Encontrados propone una plataforma que permita publicar objetos perdidos o encontrados, realizar búsquedas por categoría y facilitar el contacto entre las partes involucradas. Esta solución mejora la tasa de recuperación de pertenencias, reduce costos de reemplazo y optimiza el tiempo dedicado a su búsqueda.
